The Cycling Infrastructure Prioritisation Toolkit

3 Results

The overall result of the project is a prototype national map for cycling infrastructure prioritisataion, in-line with the initial aim of the project. As described in the CyIPT Manual and shown on the main panel at www.cyipt.bike, there are 9 layers available to the user providing an unprecedented level of data access of relevance to cycle planning in a single place.
